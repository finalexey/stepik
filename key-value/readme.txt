Реализация key-value storage. Задача - написать скрипт, который принимает в качестве аргументов ключи и значения и выводит информацию из хранилища (в данном случае - из файла).

Запись значения по ключу:
> storage.py --key key_name --val value

Получение значения по ключу:
> storage.py --key key_name

Ответом в данном случае будет вывод соответствующего значения >value.

Метрики сохраняются в порядке их добавления. Если значений по ключу не было найдено, вывести пустую строку или None.

Для работы с аргументами командной строки используется модуль argparse. Необходимо считать аргументы, переданные программе, и записать соответствующую пару ключ-значение в файл хранилища или вывести значения, если был передан только ключ. Данные хранятся в формате JSON с помощью стандартного модуля. Файл создается с помощью модуля tempfile.
